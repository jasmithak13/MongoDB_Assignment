# MongoDB_Assignment

# About This Assignment
This assignment contains 30 MongoDB problems covering basic to advanced 
database operations like CRUD, aggregation, indexing, and array queries.

# File Structure
mongodb-assignments/
├── problem1_coffee_shop.js
├── problem2_contact_book.js
├── problem3_todo_list.js
├── problem4_expense_tracker.js
├── problem5_movie_watchlist.js
└── ... (up to problem30)

# Topics Covered
- insertOne() and insertMany()
- find() with filters and projection
- updateOne() and updateMany()
- deleteOne() and deleteMany()
- Aggregation ($group, $avg, $sum, $unwind)
- Indexing (single field and multikey)
- Array query operators ($size, $elemMatch)

# How to Run
1. Install MongoDB on your system
2. Open MongoDB Shell (mongosh)
3. Run each .js file using:
   load("problem1_coffee_shop.js")
